# Shashi Sunder Rajan — Product Build Portfolio

Static GitHub Pages portfolio for public builds and product case studies.

## Repository name

Create a public repository named:

ssunderrajan.github.io

## Files to upload

Upload these files to the repository root:

- index.html
- style.css
- projects.js
- assets/

## Public URL

After GitHub Pages is enabled, the site will be available at:

https://ssunderrajan.github.io

## How to update later

Add future projects under one of these categories:

- Live Data & Geospatial Intelligence
- Compliance, KYC & Risk Systems
- AI & Document Intelligence
- Telco & Communications APIs
- Product Strategy Builds

## Confidentiality rule

Do not publish employer source code, screenshots, internal architecture, customer data, credentials, commercial details, or non-public regulatory interpretations.

Use sanitized case studies for enterprise work.
